## Commands to Run (After Creating GitHub Repository)

After you create the repository on GitHub (https://github.com/new), run these commands:

```bash
# Add the remote repository
git remote add origin https://github.com/vikingdevx/valentines-day.git

# Push your code to GitHub
git push -u origin main
```

## Enable GitHub Pages

1. Go to your repository: https://github.com/vikingdevx/valentines-day
2. Click on **Settings** (top menu)
3. Click on **Pages** (left sidebar)
4. Under "Source", select **main** branch
5. Click **Save**
6. Wait 1-2 minutes for deployment
7. Your website will be live at: https://vikingdevx.github.io/valentines-day/
